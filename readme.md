# 🚀 Space Invaders (PPlay)

## 📌 Descrição

Este projeto é uma implementação do clássico **Space Invaders**, desenvolvida em Python utilizando a biblioteca **PPlay**. O jogo conta com um sistema de menus interativos, seleção de dificuldade e base para expansão com mecânicas completas de gameplay.

---

## 🎮 Funcionalidades atuais

* Menu principal com botões:

  * Jogar
  * Dificuldade
  * Ranking
  * Sair
* Navegação entre telas usando mouse
* Sistema de estados (menu, jogo, dificuldade)
* Interface gráfica com sprites personalizados

---

## 🛠️ Tecnologias utilizadas

* Python
* PPlay
* Pygame (base do PPlay)
